# toLowerCase()

## Descripció

Obteniu una versió en minúscules d'una cadena. A partir de l'1.0, toLowerCase() modifica la cadena al seu lloc en lloc de tornar-ne una de nova.

## Sintaxi

`myString.toLowerCase()`

## Paràmetres

`myString`: una variable de tipus String
