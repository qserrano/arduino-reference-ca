# toUpperCase()

## Descripció

Obteniu una versió en majúscules d'una cadena. A partir de l'1.0, toUpperCase() modifica la cadena al seu lloc en lloc de tornar-ne una de nova.

## Sintaxi

`myString.toUpperCase()`

## Paràmetres

`myString`: una variable de tipus String
