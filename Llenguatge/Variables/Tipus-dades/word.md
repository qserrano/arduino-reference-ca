# word

## Descripció

Una paraula pot emmagatzemar un nombre sense signe d'almenys 16 bits (de 0 a 65535).

## Sintaxi

`word var = val;`

## Paràmetres

`var`: nom de variable.  
`val`: el valor a assignar a aquesta variable.

## Exemple de codi

`word w = 10000;`
