# byte

## Descripció

Un byte emmagatzema un número sense signe de 8 bits, de 0 a 255.

## Sintaxi

`byte var = valor;`

## Paràmetres

`var`: nom de la variable.  
`val`: el valor a assignar a aqueixa variable.
