# bitRead()

### Descripcio

Llegeix un bit d'un número.


### Sintaxi

`bitRead(x, n)`


### Paràmetres

`x`: el nombre a partir del qual cal llegir.  
`n`: quin bit cal llegir, començant per 0 per al bit menys significatiu (l'extrem dret)


### Devolucions

El valor del bit (0 o 1).
