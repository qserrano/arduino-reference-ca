# lowByte()

### Descripcio

Extreu el byte d'ordre inferior (l'extrem dret) d'una variable (per exemple, una paraula).

### Sintaxi

`lowByte (x)`


### Paràmetres

`x`: un valor de qualsevol tipus


### Devolucions

Tipus de dades: byte.
