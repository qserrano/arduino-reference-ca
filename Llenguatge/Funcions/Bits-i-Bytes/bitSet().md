# bitSet()

### Descripcio

Estableix (escriu un 1 a) un bit d'una variable numèrica.


### Sintaxi

`bitSet(x, n)`


### Paràmetres

`x`: la variable numèrica el bit de la qual cal establir.  
`n`: quin bit cal establir, començant a 0 per al bit menys significatiu (l'extrem dret)
