# highByte()

### Descripcio

Extreu el byte d'ordre superior (l'extrem esquerre) d'un tipus de dades word
(o el segon byte més baix d'un tipus de dades més gran).


### Sintaxi

`highByte (x)`


### Paràmetres

`x`: un valor de qualsevol tipus


### Devolucions

Tipus de dades: byte.
