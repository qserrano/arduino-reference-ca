# bit()

### Descripcio

Calcula el valor del bit especificat (el bit 0 és 1, el bit 1 és 2, el bit 2 és 4, etc.).


### Sintaxi

*  `bit(n)`


### Paràmetres

*  `n`: el bit el valor del qual cal calcular


### Devolucions

El valor del bit.
